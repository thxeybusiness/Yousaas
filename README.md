# You'SaaS

Site vitrine de **You'SaaS** — studio SaaS nouvelle génération.
On conçoit, développe et fait scaler des produits SaaS de bout en bout.

## Stack
Site statique (HTML/CSS/JS), déployé sur **Vercel**.

## Développement local
Ouvrir `index.html` dans un navigateur, ou servir le dossier :

```bash
npx serve .
```

## Déploiement
Chaque push sur `main` déclenche un déploiement automatique sur Vercel.
Domaine de production : https://yousaas.business
